How to change Game name?

  1. Open Android Studio > > res > values > strings.xml

  2. Enter your app name inside "app_name" string tag:
    <string name="app_name">Slowdown</string>


How to change AdMob Ad id's?


Open Android Studio > > res > values > strings.xml

  1. Enter your AdMob App ID inside "_app_id" string tag:
    <string name="_app_id">ca-app-pub-3940256099942544~3347511713</string>

  2. Enter your AdMob Banner ID inside "_banner_id" string tag:
    <string name="_banner_id">ca-app-pub-3940256099942544/6300978111</string>

  3. Enter your AdMob Interstitial ID inside "_interstitial_id" string tag:
    <string name="_interstitial_id">ca-app-pub-3940256099942544/1033173712</string>

  4. Enter your AdMob Reward Video ID inside "_reward_video_id" string tag:
    <string name="_reward_video_id">ca-app-pub-3940256099942544/5224354917</string>

How to change Facebook Ad id's?


Open Android Studio > > res > values > strings.xml

  1. Enter your Facebook Banner ID inside "_banner_id" string tag:
    <string name="_banner_id">IMG_16_9_APP_INSTALL#YOUR_PLACEMENT_ID</string>

  2. Enter your Facebook Interstitial ID inside "_interstitial_id" string tag:
    <string name="_interstitial_id">IMG_16_9_APP_INSTALL#YOUR_PLACEMENT_ID</string>

  3. Enter your Facebook Reward Video ID inside "_reward_video_id" string tag:
    <string name="_reward_video_id">VID_HD_9_16_15S_APP_INSTALL#YOUR_PLACEMENT_ID</string>

How to change More Games and Share Message of the game?


Open Android Studio > > res > values > strings.xml

  1. Enter your Google play developer link inside "more_games_url" string tag:
    <string name="more_games_url">https://play.google.com/store/apps/dev?id=5924850577512930216</string>

  2. Edit your share message inside "share_message" string tag:
    <string name="share_message">OMG been playing # for hours!, I scored</string>

How to change Splash Screen

  1. Open Android Studio > app > assets > default.png

  2. Replace default.png with your splash screen image, remember to name it default.png, also keep the dimensions of the replaced image, same as old image.

How to change App Icon

  1. Open Android Studio > app > res > drawable > icon.png

  2. Replace icon.png with your icon image, remember to name it icon.png
